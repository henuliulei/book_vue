#项目介绍
使用element-ui和vue框架实现的管理系统的前端。
运行效果展示：     <br>
相对应博客:   <br>
后端程序:[github代码](https://github.com/henuliulei/ssm_bookSystem)   <br>
#使用技术
*   element-ui 
*   vue
*   html+css+js
*   axios vuex  vue-router core-js
#运行环境
*   vue2 
*   vue cli
>在终端输入 vue ui打开可视化终端在网页运行项目,同时保证后端程序在运行
